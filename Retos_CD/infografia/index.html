Dos Niveles de Clasificación

Nivel 1 · Tareas

¿Qué hace el modelo?

Define la naturaleza del problema que el modelo resuelve. Cada algoritmo se ubica según su uso más frecuente.

Clasificación
Decision Tree, Naive Bayes, Reg. Logística, SVM, XGBoost
Regresión
Lineal, Polinomial, MLP
Clustering
K-means, DBSCAN, BIRCH
Asociación / Recomendación
Apriori, FP-Growth, Filtrado Colaborativo
Nivel 2 · Dominios Transversales

¿Dónde cruzan las tareas?

No compiten con el Nivel 1; lo cruzan. Un dominio emplea múltiples tareas.

T
Minería de Textos
Clasificación de sentimientos (BETO, RoBERTa), clustering de documentos (TF-IDF + K-means).
S
Series de Tiempo
Predicción con ARIMA/Prophet (regresión), detección de anomalías con LSTM (clasificación).
N
Redes Neuronales
BETO/RoBERTa (Transformer → clasificación), MLP (regresión), Autoencoders + K-means (clustering).

Casos de Estudio

Caso A · Clasificación Tabular

Predicción de Resiliencia Académica en Zonas Afectadas por el Conflicto

1
Preprocesamiento
1.6M+ registros ICFES. StandardScaler, OneHotEncoder, 62 variables. Undersampling 60/40 para desbalance (76% vs 24%).
2
Modelos
Reg. Logística (línea base), SVM lineal, XGBoost (grilla 48 comb., scale_pos_weight), MLP (dropout + batch norm).
3
Evaluación
AUC-ROC, Recall, Precision, F1, matrices de confusión, curvas ROC. XGBoost 60/40: AUC 0.85, F1 0.62. MLP: recall ~0.76.
4
Despliegue
Sistema predictivo para identificar estudiantes resilientes en contextos vulnerables e intervención educativa.
Caso B · NLP / Minería de Textos

Percepción Mediática de la Reforma a la Salud en Colombia

1
Preprocesamiento
Web scraping de 1 401 noticias (2022–2024). Tokenización, lematización, eliminación de stopwords/emojis. Filtro IQR → 1 362 noticias.
2
Modelos
Fine-tuning de BETO y RoBERTa (Transformers encoder). ChatGPT-4o como contraste. Similitud: TF-IDF, Doc2Vec, MPNet.
BETORoBERTa
Transformers encoder fine-tuneados para clasificación de sentimiento (NEG/NEU/POS).
BETO → F1: 0.91 · RoBERTa → F1: 0.89
3
Evaluación
Precision, Recall, F1, Accuracy por clase. Etiquetado manual del 26.43% del corpus. ChatGPT-4o como contraste: 67.3% accuracy.
4
Despliegue
Dashboard Power BI interactivo con distribución de sentimiento por región (Andina, Caribe, Pacífica).
